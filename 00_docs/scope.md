# Scope

## In scope (v0)
- Snapshot-based inventory (JSON exports)
- Hygiene assessment:
  - tagging coverage (owner/costCenter/env)
  - diagnostic settings coverage (high-level)
  - public exposure signals (e.g., public IP / public access flags)
- Findings + plan (small, actionable)

## Out of scope (v0)
- Automatic remediation
- Full cost calculation feom billing APIs
- Full security posture management
